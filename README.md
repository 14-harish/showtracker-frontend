# ShowTracker Frontend

Frontend for the ShowTracker application.

## Tech Stack
- React
- Tailwind CSS
- JavaScript

## Features
- Login and signup
- Search movies and TV shows
- Filter by year and type
- Watchlist and progress tracking
- Poster modals and dashboard view

## Setup

### 1. Clone repository

git clone https://github.com/14-harish/showtracker-frontend.git
cd showtracker-frontend

2. Install dependencies
npm install

3. Start development server
npm start

Frontend runs at:
http://localhost:3000

Backend Requirement

Backend must be running at:
http://127.0.0.1:8000

API base URL is configured in:
src/api/client.js
